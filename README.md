# Income Tax Calculator

A simple and interactive Income Tax Calculator built using **HTML**, **CSS**, and **JavaScript**. This project allows users to enter their annual income and instantly calculate the tax payable according to predefined tax slabs.

## Features

* User-friendly interface with a modern dark theme.
* Responsive and clean design.
* Instant tax calculation using JavaScript.
* Real-time display of total tax amount.
* Form reset after calculation.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

1. Enter your annual income in the input field.
2. Click the **Calculate Income** button.
3. The application calculates the tax based on the specified tax slabs.
4. The total tax amount is displayed on the screen.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/itzSurbhi/income-tax-calculator.git
```

2. Navigate to the project folder:

```bash
cd income-tax-calculator
```

3. Open `index.html` in your browser.
